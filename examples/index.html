Spotter Examples

Here's a couple of examples that use spotter.

  • The Bieber Example: This example alerts the user at 10 second intervals when there's a new tweet about Justin Bieber (which, at the time of this writing, is at least every 10 seconds). The entire program (with the exception of Spotter) is in the source document.
  • The Gadget Example: This is example is slightly more interesting; it mimics the commonly seen Twitter gadget. The current search term is 'oil spill.' The JavaScript code can be found here. This example uses the JQuery library for selectors, animation, and insertions into the DOM.
