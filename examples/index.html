Here's a few examples that use spotter.

  • The Bieber Example: This example alerts the user at 10 second intervals when there's a new tweet about Justin Bieber (which, at the time of this writing, is at least every 10 seconds). The entire program (with the exception of Spotter) is in the source document.
  • The Trends Example: In this example, the trending topics are added and then whenever they change the page is updated. The Spotter checks with the Twitter API every 30 seconds. This example uses the JQuery library for selectors, animations and insertions into the DOM.
  • The Gadget Example: This is example is slightly more interesting; it mimics the ubiquitous Twitter gadget. The current search term is 'oil spill.' The JavaScript code can be found here. This example also uses the JQuery library.
